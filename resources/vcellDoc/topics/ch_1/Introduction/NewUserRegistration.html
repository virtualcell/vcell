New User Registration

A user trying to use the Virtual Cell for the first time is prompted to register.



The Virtual Cell User Registration dialog prompts the user to choose a login id and password. In order to complete the registration, the user needs to provide other information such as a valid email address (for VCell software notifications and to retrieve forgotten passwords), first and last name of user, job title, organization and address. The user can choose to decline receiving emails regarding VCell notifications by clicking on the checkbox at the bottom of the registration dialog.
