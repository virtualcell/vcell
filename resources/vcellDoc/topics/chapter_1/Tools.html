Tools

The Tools menu bar option allows the user to launch tools that can be used to model in the Virtual Cell.



Clicking the Tools menu shows the possible menu item selections. Selecting

  • Launch BioNetGen - launches the BioNetGen software.
  • Launch Virtual FRAP - launches the Virtual FRAP software.
  • Transform to Stochastic Capable - launches a tool to convert a deterministic model to a stochastic-capable model, if possible.
